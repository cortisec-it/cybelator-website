Lab Infrastructure
Hands-on Environment Design
60% of program time is labs. Infrastructure is a hybrid of physical gear, local VMs, and cloud-based practice environments — keeping costs manageable without sacrificing realism.
⬡

Core Virtualization Lab (Physical)

1× server (32–64 GB RAM, 8-core CPU, 2TB NVMe) running Proxmox or VMware ESXi. Hosts 8–12 VMs simultaneously — Windows Server AD, pfSense/OPNsense firewall, Kali, Ubuntu, and Windows 10 clients. Shared among batch of 15–20 via thin provisioning.

⬡

Firewall Lab

Check Point CloudGuard trial (AWS/Azure free tier) + Fortinet FortiGate VM trial + Sophos UTM Home edition. Students configure real policies, NAT rules, IPS signatures, and site-to-site VPNs. Each student gets a personal isolated tenant.

⬡

SIEM Lab — Splunk, Sentinel, QRadar

Splunk Free (500MB/day) deployed locally. Microsoft Sentinel in Azure free trial. IBM QRadar Community Edition. Students ingest real firewall logs, Windows event logs, and simulated threat feeds and write detection rules.

⬡

SOC Simulation — Ticketing & Alert Triage

Freshservice free tier (or osTicket self-hosted) as the ticketing system. A custom-built alert injector feeds simulated alerts into students' queues. Students practice triage, escalation writing, and ticket closure documentation.

⬡

Cloud Security Lab — AWS & Azure

AWS Educate accounts + Azure for Students (₹0 cost). Students configure Security Groups, IAM policies, Azure AD Conditional Access, and review CloudTrail/Activity Log alerts. Focus on misconfiguration detection.

⬡

EDR Lab — SentinelOne, Defender

SentinelOne trial tenant for the batch. Microsoft Defender for Endpoint via Microsoft 365 Developer Program (free). Students enroll endpoints, review detections, and practice threat isolation workflows.

⬡

Monitoring Lab — Zabbix, PRTG, Grafana

Zabbix Community (free, self-hosted), PRTG free for up to 100 sensors, Grafana OSS with Prometheus. Students build dashboards for network health, set alert thresholds, and simulate performance degradation scenarios.

⬡

Remote Lab Access

Students access the lab VM environment 24/7 via Apache Guacamole (browser-based RDP/VNC). No need for high-end student hardware — a laptop with a browser and 4 Mbps internet is sufficient.

Format: Mon–Fri, 9 AM–5 PM (full-time) or Sat–Sun intensive (part-time option). Conducted at Cortisec's Noida office or remotely with supervised lab sessions. Maximum 5 interns per batch for quality.

Market Analysis · Pricing Strategy
Assam & Northeast India Market
The cybersecurity training market in Northeast India is significantly underserved. Guwahati has a growing student population (GU, Assam Engineering College, GIMT, Don Bosco, Royal Global) with very limited practical cybersecurity program options. Most existing institutes offer CEH/CHFI certification prep only, with no enterprise-practical orientation.
Market gap: No program in Assam currently offers enterprise firewall exposure (Check Point, Palo Alto), real SOC simulation, SIEM hands-on, or a structured internship pipeline. This is a first-mover opportunity. Comparable programs in Bengaluru/Hyderabad cost ₹60,000–₹1,20,000 and still lack the practical depth described here.

Hiring Readiness
Target Roles & Salary Map
Entry-level cybersecurity roles for fresh graduates in India. These are realistic placements achievable within 3–6 months post-program completion.

SOC Analyst L1

₹3.5 – ₹5.5 LPA

Alert monitoring, ticket handling, log review. Most common entry-level role. High demand at MSSPs and large enterprises.

Security Operations Associate

₹4 – ₹6 LPA

Broader ops role — firewall, endpoints, identity. Common at IT services firms like Wipro, HCL, Tech Mahindra.

Firewall Support Engineer

₹3.5 – ₹5 LPA

Check Point / Fortinet / Palo Alto operation. Often outsourced to MSSPs. Strong demand from banking and telecom sectors.

Network Security Engineer

₹4 – ₹7 LPA

Network + security combined. Routers, firewalls, VPN. ISPs, data centers, enterprise IT teams.

IAM Support Analyst

₹3.5 – ₹5.5 LPA

Identity lifecycle, access requests, MFA support. Growing role as Zero Trust adoption increases in Indian enterprises.

Security Monitoring Analyst

₹4 – ₹6 LPA

SIEM-focused. Write detection rules, investigate alerts. BFSI and large IT companies are biggest hirers.

NOC/SOC Engineer

₹3 – ₹5 LPA

Hybrid network + security operations. Common at telecom companies, data centers, managed service providers.

Cybersecurity Support Engineer

₹3.5 – ₹5.5 LPA

Customer-facing technical support for security products. Vendors like Check Point, Fortinet, SentinelOne hire for this.
